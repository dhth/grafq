`gcue` (stands for "graph query") lets you query Neo4j/AWS Neptune databases via
an interactive console.

It is intended to be a Rust port of [graphc].

> [!NOTE]
> gcue is alpha software. Its interface and behaviour might change in the near
> future.

[graphc]: https://github.com/dhth/graphc
